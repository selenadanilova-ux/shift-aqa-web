# Интенсив ШИФТ — Практические задания

Репозиторий для выполнения домашних работ и практических заданий в рамках интенсива ШИФТ.  
Все проекты реализованы на **TypeScript**.

## 🛠️ Стек технологий

* **Язык программирования:** TypeScript
* **Платформа запуска:** Node.js
* **Фреймворк для автотестов:** Playwright
* **Среда разработки (IDE):** Visual Studio Code
* **Система контроля версий:** Git / GitHub

В этом модуле осуществляется настройка окружения для автоматизации визуального тестирования интерфейсов.

## 🛠️ Настройка окружения

Для запуска проекта локально выполните следующие шаги:

### 1. Предварительные требования
Убедитесь, что на вашем компьютере установлены:
* **Git** (версии 2.x или выше)
* **Node.js** (LTS версия)
* **Visual Studio Code** (с плагинами для TypeScript и автотестов)

### 2. Клонирование репозитория
Склонируйте проект на локальный компьютер:
```bash
git clone https://github.com
cd shift-aqa-web
```

### 3. Сборка и подготовка окружения
 Разверните окружение согласно инструкции из канала «Автотестирование»:
```bash
npm install
```

## 🧪 Запуск автотестов (Playwright)

Все команды выполняются во встроенном терминале VS Code.

### 1. Запуск всех тестов в консоли
```bash
npx playwright test
```

### 2. Запуск тестов в режиме UI (с визуальным интерфейсом)
```bash
npx playwright test --ui
```

### 3. Просмотр отчета о тестировании (HTML Report)
Если тест упал или вы хотите посмотреть детальный лог выполнения:
```bash
npx playwright show-report
```
